# Esbuild

Use [esbuild](https://github.com/evanw/esbuild) from Ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'esbuild'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install esbuild
